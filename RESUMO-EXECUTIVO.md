## 📌 O QUE É ESTE PROJETO?

Uma **revisão sistemática de literatura** sobre:

> **"Transferência de Conhecimento Externo, Capacidade Absortiva e Competitividade em MPEs Brasileiras"**

Seguindo rigorosamente os princípios de **Open Science** e protocolo **PRISMA**.

---

## ✨ POR QUE É ESPECIAL?

Este projeto não é uma revisão comum. É:

✅ **Open Science:** Tudo documentado, replicável, transparente  
✅ **Automatizado:** Scripts Python com IA (Gemini / Claude)  
✅ **Estruturado:** 7 teorias integradas em um framework  
✅ **Validado:** Validação humana de 10-20% da amostra  
✅ **Rastreável:** Git + versionamento de todas as decisões  
✅ **Profissional:** Pronto para publicação e futuras pesquisas

---

## 🎯 OBJETIVOS PRINCIPAIS

### Objetivo Geral
Identificar lacunas de pesquisa sobre como transferência de conhecimento externo, mediada por capacidade absortiva, impacta a competitividade de MPEs brasileiras.

### Objetivos Específicos
1. **Mapear** a transferência de conhecimento externo em MPEs
2. **Entender** como capacidade absortiva medeia este processo
3. **Analisar** efeitos na competitividade
4. **Identificar** lacunas críticas de pesquisa
5. **Gerar** proposições testáveis para dissertação
